Sie bekommen im Browser den HTTP Status Code 400. Was hat das zu bedeuten? Was können Sie dagegen tun

Der 400 Bad Request Error ist ein HTTP-Response-Statuscode. die anzeigt, dass der Server die vom Client gesendete Anforderung aufgrund falscher Syntax, ungültigem Request-Nachrichten-Rahmen oder irreführendem Request-Routing nicht verarbeiten (verstehen) konnte. Lösung: Eingaabe ändern

Sie bekommen im Browser den HTTP Status Code 403. Was hat das zu bedeuten? Was können Sie dagegen tun?

Der Zugriff auf die angeforderte Ressource ist verboten. Der Server hat die Anfrage verstanden, wird sie aber nicht erfüllen. Lösungsansatz: Cache löschen

HTML-Literatur lesen und Fragen beantworten

Was ist HTML?

HTML steht für HyperText Markup Language. Es handelt sich um eine markup-Sprache zur Programmierung von Websites. HTML beschreibt die Struktur einer Website. HTML besteht aus Elementen, die wiederum dem Browser spezifizieren, wie eine Website angezeigt werden soll.

Wie kann man eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen?

<ol> <li>Erstes Element der Ebene 1 <ol> <li>Erstes Element der Ebene 2 <ol> <li>Erstes Element der Ebene 3</li> <li>Zweites Element der Ebene 3</li> <li>Drittes Element der Ebene 3</li> </ol> </li> <li>Zweites Element der Ebene 2</li> <li>Drittes Element der Ebene 2</li> </ol> </li> <li>Zweites Element der Ebene 1</li> <li>Drittes Element der Ebene 1</li> </ol>

Wie ist eine HTML-Tabelle aufgebaut?

<table> dient als Container für die gesamte Tabelle. <tr> tr steht für table row <th>Spaltenüberschrift 1</th> <th>Spaltenüberschrift 2</th> <th>Spaltenüberschrift 3</th> </tr> <tr> <td>Zelle 1, Zeile 2</td> td steht für table data <td>Zelle 2, Zeile 2</td> <td>Zelle 3, Zeile 2</td> </tr> <tr> <td>Zelle 1, Zeile 3</td> <td>Zelle 2, Zeile 3</td> <td>Zelle 3, Zeile 3</td> </tr> </table>

Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?

Groß und Kleinschreibung, relativer oder absoluter Pfad, Dateiendungen

Wie baut man in HTML ein Menü?

Mithilfe einer ungeordneten Liste und dem href-Attribut des a-Tags lässt sich ein Menü schnell umsetzen

Welche Attribute sollte man bei Bildern wie verwenden?

Das src-Attribut definiert die Bildquelle und das alt-Attribut liefert eine alternative Textbeschreibung, die das Bild für sehbehinderte Benutzer erklärt.
