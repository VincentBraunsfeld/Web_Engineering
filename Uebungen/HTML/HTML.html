1.2. HTTP

1. Sie bekommen im Browser den HTTP Status Code 200. Was bedeutet das?

  • Request wurde erfolgreich bearbeitet

2. Sie bekommen im Browser den HTTP Status Code 301. Was hat das zu bedeuten?

  • Der Server ist nicht merh unter dieser Adresse erreichbar

3. Sie bekommen im Browser den HTTP Status Code 400. Was hat das zu bedeuten? Was können Sie dagegen tun?

  • Der 400 Bad Request Error ist ein HTTP-Response-Statuscode. die anzeigt, dass der Server die vom Client gesendete Anforderung aufgrund falscher Syntax, ungültigem Request-Nachrichten-Rahmen oder irreführendem Request-Routing nicht verarbeiten (verstehen) konnte. Lösung: Eingaabe ändern

4. Sie bekommen im Browser den HTTP Status Code 403. Was hat das zu bedeuten? Was können Sie dagegen tun?

  • Der Zugriff auf die angeforderte Ressource ist verboten. Der Server hat die Anfrage verstanden, wird sie aber nicht erfüllen. Lösungsansatz: Cache löschen
